#### 简介

Cobotsys 3.0是[武汉库柏特科技有限公司](http://www.cobotsys.com/)发布的智能机器人操作系统，旨在实现可面向复杂应用场景和缺失硬件环境的快速构建和开发，让服务和插件可自由组合成完整的解决方案。目前支持的平台为Ubuntu16.04、Ubuntu18.04。
